h1. Was ist gdv.xport?

gdv.xport ist als Java-Bibliothek gestartet, die den Umgang mit dem GDV-Format erleichtert.
Das "x" in port steht fur "im"- und "ex"-port. Die Bibliothek ist Open Source und steht unter der Apache License.

Ab 2018 ist eine Version 3.0 ist geplant, mit der diese Bibliothek um einen Service fuer Dateien im GDV-Format erweitert wird.
Derzeitig ist folgende Aufteilung geplant:

* *lib*: gdv.xport als Java-Bibliothek
* web:
  * *service*: gdv.xport als Microservice (vermutlich als Spring-Boot-Anwendung)
  * *ui*: Web-Client zum gdv.xport-Service (Technik noch unklar, je nach vorhandene Resourcen)



h2. Aehnliche Projekte

Fuer Ruby gibt es "OpenGDV":https://github.com/vendis/opengdv/, einer Open-Source-Bibliothek, die unter der MIT-License steht.
Allerdings wurde diese Bibliothek schon seit laengerer Zeit nicht mehr aktualisiert.
